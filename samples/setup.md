# Basic Operation

```
$ git submodule add https://github.com/asami/ai-directive.git ai/directive
```

```
$ ln -s ai/directive/core/AGENT.md .
```

```
$ ln -s ai/directive/core/RULE.md .
```

# README.md

```
$ cp ai/directive/samples/README.md.sample ./README.md
```
